# Draw NMR Spectra Using Plotly

Internal backend renderer for interactive plotting.

## Usage

``` r
.draw_plotly(dat, col = NULL, ...)
```

## Arguments

- dat:

  List returned by
  [`.prep_spec()`](https://tkimhofer.github.io/metabom8/reference/dot-prep_spec.md).

- ...:

  Additional arguments passed to `plotly`.
